## 🚀 Quickstart

```bash
git clone https://github.com/KOSASIH/Singularion-Nexus.git
cd Singularion-Nexus
pip install -e "[dev]"
```

```python
from src.core.orchestrator import NexusOrchestrator, NexusConfig
import asyncio

async def main():
    async with NexusOrchestrator(NexusConfig(node_id="my-node")) as nexus:
        result = await nexus.fulfill_intent({
            "domain": "financial",
            "action": "optimize_portfolio",
            "constraints": {"risk": 0.5},
        })
        print(result)

asyncio.run(main())
```
